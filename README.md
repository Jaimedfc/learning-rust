# learning-rust
Repo to save my first rust projects! Following steps in tutorial "[The Rust Programming Language](https://doc.rust-lang.org/book/)"

## Projects

- Hello World: Simple Hello World to learn basic funtion syntax and Rust compiling using `rustc`.

- Hello Cargo: Hello World built with Rust's package manager `cargo`.

- Guessing Game: Terminal guessing game to learn about variables, matching expressions and crate management.

- Temperature Converter: Terminal Celsius <-> Fahrenheit degrees converter. Understanding variables, statements, expressions and loops.
